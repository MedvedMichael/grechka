# Getting Started with Grechka app

## A few steps to start:

1. Clone project from github;
2. Install dependencies in few steps
    1. `cd client`
    2. `yarn install`
    3. `cd ../server`
    4. `yarn install`
3. Run the project by two commands:
   1. `npm start --prefix ./server`.
   2. `npm start --prefix ./client`.
